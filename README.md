# 让VSCode像Typora一样为每个markdonw文件分文件夹存图片

![](./assets/icon.png)


## 功能介绍

- 自动计算图片哈希值，避免重复图片存储
- 根据markdown文件的名称，自动创建图片文件夹
- Typora式的图片管理体验, 完全开源
